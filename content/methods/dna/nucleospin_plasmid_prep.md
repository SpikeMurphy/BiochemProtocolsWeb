## Procedure

### Preparations

1. {{< checkbox >}} [Prepare a 5 mL culture of transformed bacteria and grow them until an OD<sub>600</sub> ~ 0.4–0.8 (best 0.6) is reached.]({{< relref "/methods/bacteria/plasmid_prep_culture.md" >}})
2. {{< checkbox >}} Optionally, pre-heat a heating block to 70 °C.
3. {{< checkbox >}} Optionally, pre-heat 60 µL of buffer AE (elution buffer) per sapmle to 70 °C.

### Harvest

1. {{< checkbox >}} Pellet bacteria @ 5,000 g for 10 minutes.
{{< timer time="0:10:00" >}}

{{< infobox type="info" title="Microfuge" >}}
⚠️ If no device for centrifuging 15 mL tubes at high enaugh force is available, a microfuge with multiple cycles can be used:

1. {{< checkbox >}} Transfer 1000 µL of the culture into a sterile 1.5 mL reaction tube.
2. {{< checkbox >}} Pellet bacteria @ 11,000 g for 30 seconds.
{{< timer time="0:00:30" >}}

3. {{< checkbox >}} Discard supernatant.
4. {{< checkbox >}} Transfer 1000 µL of the culture into a the same 1.5 mL reaction tube.
5. {{< checkbox >}} Pellet bacteria @ 11,000 g for 30 seconds.
{{< timer time="0:00:30" >}}

6. {{< checkbox >}} Discard supernatant.
7. {{< checkbox >}} Transfer 1000 µL of the culture into a the same 1.5 mL reaction tube.
8. {{< checkbox >}} Pellet bacteria @ 11,000 g for 30 seconds.
{{< timer time="0:00:30" >}}

9. {{< checkbox >}} Discard supernatant.
10. {{< checkbox >}} Transfer 1000 µL of the culture into a the same 1.5 mL reaction tube.
11. {{< checkbox >}} Pellet bacteria @ 11,000 g for 30 seconds.
{{< timer time="0:00:30" >}}

12. {{< checkbox >}} Discard supernatant.
13. {{< checkbox >}} Transfer 1000 µL of the culture into a the same 1.5 mL reaction tube.
14. {{< checkbox >}} Pellet bacteria @ 11,000 g for 30 seconds.
{{< timer time="0:00:30" >}}

{{< /infobox >}}

15. {{< checkbox >}} Discard supernatant.
16. {{< checkbox >}} Remove all access medium with a pipette.

### Cell Lysis

1. {{< checkbox >}} Add 250 µL of Buffer A1 to the pellet.
2. {{< checkbox >}} Resuspend the pellet completely by pipetting up and down and/or vortexing.
3. {{< checkbox >}} Add 250 µL of Buffer A2 (lysis buffer) to the cell suspension.
4. {{< checkbox >}} Mix by inverting the tube 10 times.
5. {{< checkbox >}} Incubate @RT for 5 minutes.
{{< timer time="0:05:00" >}}

6. {{< checkbox >}} Add 300 µL Buffer A3 (neutralization buffer).
7. {{< checkbox >}} Mix by inverting the tube 10 times.
8. {{< checkbox >}} Centrifuge the suspension @ 11,000 g for 10 minutes.
{{< timer time="0:10:00" >}}

### Loading the Column

1. {{< checkbox >}} Place a NucleoSpin Column onto a 2 mL collection tube per sample.
2. {{< checkbox >}} Provide and lable a 1.5 mL reaction tube per sample.
3. {{< checkbox >}} Add the supernatant (up to 700 µL) onto the column membrane.
4. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}

5. {{< checkbox >}} Discard the flowthrough.
6. {{< checkbox >}} Add 500 µL of Buffer AW to the column membrane.
7. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}

8. {{< checkbox >}} Discard the flowthrough.
9. {{< checkbox >}} Add 600 µL of Buffer A4 (wash buffer) to the column membrane.
10. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}

11. {{< checkbox >}} Discard the flowthrough.
12. {{< checkbox >}} Dry the membrane and remove access ethanol by centrifuging @ 11,000 g for 2 minutes.
{{< timer time="0:02:00" >}}

13. {{< checkbox >}} Discard the collection tube and place the collum onto a fresh 1.5 mL reaction tube.
14. {{< checkbox >}} Optionally, incubate the column @ 70 °C for 2 minutes to remove all residual ethanole.
{{< timer time="0:02:00" >}}

### Eluting the DNA

1. {{< checkbox >}} Add 25µL of pre-heated buffer AE (elution buffer) to the column membrane.
2. {{< checkbox >}} Close the lid and incubate @ RT for 5 minutes.
{{< infobox type="info" title="" >}}
⚠️ Optionally, incubate at 70 °C for 5 minutes instead. Note that this may reduce the final volume but can increase the concentration and overall DNA yield.
{{< /infobox >}}
{{< timer time="0:05:00" >}}

3. {{< checkbox >}} Optionally, Centrifuge @ 50 g (or the lowest force possible) for 60 seconds.
{{< timer time="0:01:00" >}}

4. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}

5. {{< checkbox >}} Add 25µL of pre-heated buffer AE (elution buffer) to the column membrane.
6. {{< checkbox >}} Close the lid and incubate @ RT for 5 minutes.
{{< infobox type="info" title="" >}}
⚠️ Optionally, incubate at 70 °C for 5 minutes instead. Note that this may reduce the final volume but can increase the concentration and overall DNA yield.
{{< /infobox >}}
{{< timer time="0:05:00" >}}

7. {{< checkbox >}} Optionally, Centrifuge @ 50 g (or the lowest force possible) for 60 seconds.
{{< timer time="0:01:00" >}}

8. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}

{{< infobox type="info" title="" >}}
⚠️ You can do an additional third elution to obtain more overall DNA mass. If concentration is crutial, use another reaction tube for the additional elution.

1. {{< checkbox >}} Optionally, place the column on another reaction tube.
2. {{< checkbox >}} Add 10-25µL of pre-heated buffer AE (elution buffer) to the column membrane.
3. {{< checkbox >}} Close the lid and incubate @ 70 °C for 5 minutes.
4. {{< checkbox >}} Optionally, Centrifuge @ 50 g (or the lowest force possible) for 60 seconds.
{{< timer time="0:01:00" >}}

5. {{< checkbox >}} Centrifuge @ 11,000 g for 60 seconds.
{{< timer time="0:01:00" >}}
{{< /infobox >}}

9. {{< checkbox >}} Discard the collumn and store the eluate @ -20 °C for long lime storage.
